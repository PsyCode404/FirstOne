Introduction

Why Python?

Python is a popular programming language created by Guido van Rossum in 1991. It is versatile and used for web development, software development, mathematics, and system scripting.

What Can Python Do?

  • Create web applications on a server.
  • Handle workflows and connect to databases.
  • Analyze big data and perform complex mathematics.
  • Support rapid prototyping and production-ready software development.
Python Syntax

Introduction to Python Syntax

Python is known for its simple and readable syntax. Here are some key points:

1. Indentation

Python uses indentation to define code blocks:

if 5 > 2:
    print("Five is greater than two!")

2. Variables

Variables are dynamically typed:

x = 10
y = "Hello"
print(x, y)

3. Comments

Use `#` for single-line comments:

# This is a comment
print("Python is awesome!")
Python Operators

Python Operators

Operators in Python are used to perform operations on variables and values:

  • Arithmetic: +, -, *, /
  • Comparison: ==, !=, >, <
  • Logical: and, or, not
  • Assignment: =, +=, -=
Python While Loop

Using While Loops

While loops execute a block of code as long as the condition is true:

x = 1
while x < 5:
    print(x)
    x += 1
Python For Loop

Using For Loops

For loops iterate over a sequence (like a list or string):

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
Python Dictionaries

Introduction to Dictionaries

Dictionaries store data in key-value pairs:

my_dict = {
    "name": "John",
    "age": 30
}
print(my_dict["name"])
Python Sets

Introduction to Sets

Sets are unordered collections with no duplicate elements:

my_set = {"apple", "banana", "cherry"}
my_set.add("orange")
print(my_set)
